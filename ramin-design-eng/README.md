# ramin-design-eng

A design engineering skill for React and Next.js — covering animation, form UX, data tables, and mobile-first development.

Built by [Ramin Tahbaz](https://ramintahbaz.com).

## Install

```bash
npx skills add rtahbaz/ramin-design-eng
```

## What it covers

- Animation decision framework (should it animate, easing, duration)
- Form UX and error state patterns
- Data tables and dashboard design
- Mobile-first and low-end device considerations
- Component principles (buttons, popovers, tooltips)
- Performance rules for React / Next.js
- Accessibility (reduced motion, touch targets, iOS input quirks)

## Philosophy

Interfaces should disappear. The best UI is the one the user never thinks about — they came to do something, and they did it. Every decision in this skill exists to remove friction between intention and completion.

Clarity over decoration. Details compound.

## Usage

Once installed, Claude Code will apply these principles automatically when you ask it to build, review, or polish any frontend interface.

```bash
# Review existing UI code
"review this component using the ramin-design-eng skill"

# Build something new
"build a payment confirmation form"

# Get a specific critique
"audit the animation in this drawer"
```
